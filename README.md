# MG5795 Genomic Data Analysis – Final Project
**The Ohio State University | Autumn 2025**

This project was completed as the final assignment for MG5795: Genomic Data Analysis.

## Project Overview
Analysis was conducted across three components:

- **Gene Annotations Analysis** – Unix-based parsing of maize and human GTF files to compare gene content (protein-coding, miRNA, lncRNA); custom BED file generation
- **Gene Annotations and Comparisons** – Ab initio gene annotation using Helixer, TE annotation using EDTA, structural variant calling with nucmer/Assemblytics/SURVIVOR and minimap2/Sniffles, and CpG methylation analysis of PacBio HiFi reads
- **RNA-Seq Data Analysis** – Full pipeline including FastQC, Trimmomatic, STAR alignment, and DESeq2 differential expression analysis in R

## Tools & Languages
Unix | R (DESeq2, ggplot2) | STAR | FastQC | Trimmomatic | Helixer | EDTA | bedtools | minimap2 | Sniffles | IGV
